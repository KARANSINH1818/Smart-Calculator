## 📌 Project Description

This project is an advanced web-based calculator designed with a clean modern UI.  
It performs arithmetic calculations while maintaining a history of operations and supporting keyboard input for faster usage.

---

## 🚀 Features

- Perform basic arithmetic operations (+, −, ×, ÷)
- Calculation history tracking
- Clear history functionality
- Keyboard input support
- Error handling for invalid expressions
- Responsive design for mobile and desktop
- Modern dark-themed user interface
- Interactive button hover effects

---

## ⭐ Advanced Features

- Backspace support using keyboard
- Scrollable history panel
- Clean and intuitive layout
- Fast real-time calculations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 🎯 Future Improvements

- 🧮 Scientific calculator functions (sin, cos, tan, √, power)
- 📊 Save calculation history permanently (Local Storage)
- 🎤 Voice input support for calculations
- 📋 Copy result to clipboard
- 🎨 Light/Dark theme toggle with animation
- 📱 Mobile app–style UI improvements
- 🧠 Expression validation for complex calculations

## 🎯 How to Use

1. Open the calculator in your browser  
2. Click buttons or use keyboard to enter values  
3. Press "=" or Enter to calculate  
4. View previous calculations in the history panel
